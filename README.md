# Onshape Desktop App
Run Onshape natively with hardware acceleration built-in.

#### Building Locally:
- Install: `npm i`
- Package: `npm run package`
- Release available in `./onshape/out/`

#### Running:
- Download the release for the intended platform
- Run the file
